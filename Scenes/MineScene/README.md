# 矿井场景 (Mine Scene)

## 功能
- 地形生成系统
- 挖掘机制
- 资源收集
- 玩家移动和交互

## 文件说明
- `MineScene.tscn` - 场景文件
- `MineScene.gd` - 场景脚本

## 地形系统
- **土壤层** (0-10深度): 基础挖掘材料
- **石头层** (10-20深度): 较硬材料
- **矿物层** (20+深度): 珍贵资源

## 资源类型
- 泥土 (1金币)
- 石头 (2金币)
- 煤炭 (5金币)
- 铁矿 (10金币)
- 金矿 (25金币)
- 钻石 (50金币)

## 控制
- **ESC**: 返回城镇
